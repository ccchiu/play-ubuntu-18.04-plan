
# Prototype play-icewm

## Usage


### install

``` sh
$ ./install.sh
```

or

``` sh
$ make install
```


### package-install

``` sh
$ ./package-install.sh
```

or

``` sh
$ make package-install
```


### config-install

``` sh
$ ./config-install.sh
```

or

``` sh
$ make config-install
```


## Keyboard shortcuts

Please refer to

* $ less ~/.icewm/keys
* $ man icewm-keys
* $ man icewm | grep 'KEYBOARD SHORTCUTS' -A 237 | less
